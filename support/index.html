Документация во Flash CS3
Для подключения документации скопируйте содержимое архива (.zip | 454kb) в каталог
C:\Documents and Settings\All Users\Application Data\Adobe\Flash CS3\en\Configuration\HelpPanel\Help\Alternativa3D

Так же документация может быть подключена к Eclipse в виде плагина. Плагин можно установить из Eclipse, указав в качестве сайта обновлений адрес http://help.alternativaplatform.com
Eclipse 3.4: Help > Software Updates > Available Software > Add Site…
