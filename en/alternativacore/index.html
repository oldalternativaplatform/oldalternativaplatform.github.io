AlternativaCore is a cluster of several Linux-powered Java servers. It is a ready to use core for creating real-time and turn-based multiplayer online games. Due to its orientation on applied programming, the platform allows developers to save time and money.

Main Features

Resource System
Version control, resource editing, caching, dependence resolving and traffic balancing.

Net Protocol
Orientation to minimal packet size and high speed data transfer.

Cluster
Distribution of a game world to multiple servers using a load balancing system.

Distributed NoSQL-DB
Allows storage of an almost unlimited amount of data.

Web-based project administration system
Manage all necessary parameters of game objects easily.

Automatic Client Code Generation
Allows easier and faster development of client side.

Modular OSGi core
Allows expansion to suit different needs.

Maven-based Project Management System
Decreases expenses on managing large products.

Billing
Supports payment systems utilized by social networks and online games.

Social Networks Support
Platform is ready to work with social networks, including Facebook.

External Authorization Support
Social networks, Yandex, Gmail, LiveJournal and OpenID.

The server side is successfully used in browser-based real-time 3D action game Tanki Online, including its Chinese version.
